# Module 11: নিরাপত্তা ও নৈতিকতা (Safety & Ethics)

এআই খুবই শক্তিশালী, কিন্তু এটি যদি ভুল হাতে পড়ে বা ভুল কথা বলে, তবে তা সমাজের জন্য ক্ষতিকর হতে পারে। আজ আমরা শিখবো কীভাবে আপনার বানানো চ্যাটবট বা এআই এজেন্টকে নিরাপদ রাখা যায়। আমরা কোনো থার্ড-পার্টি টুল ছাড়া একটি 'Safety Layer' বানাবো।

## ১. এআই-এর ঝুঁকিগুলো কী কী?
*   **Harmful Output:** বিপজ্জনক কাজ করার পরামর্শ দেওয়া।
*   **Hallucination:** তথ্য না জেনেও আত্মবিশ্বাসের সাথে ভুল তথ্য দেওয়া।
*   **Bias:** কোনো নির্দিষ্ট ধর্ম, বর্ণ বা লিঙ্গের প্রতি বৈষম্যমূলক কথা বলা।

## ২. গার্ডরেইল (Guardrails) কী?
এটি হলো আপনার মডেলের চারপাশের একটি অদৃশ্য দেয়াল। এটি ইনপুট এবং আউটপুট চেক করে ফিল্টার করে দেয়।

## 💡 বাস্তব জীবনের উদাহরণ (Analogy)

ভাবুন একটি স্কুলে একজন দারোয়ান আছে।
সে চেক করে দেখে কেউ ক্লাসে প্রবেশের সময় কোনো ক্ষতিকর বস্তু নিয়ে আসছে কি না। আবার ছুটির সময় সে নিশ্চিত করে কোনো ছোট বাচ্চা ভুল মানুষের সাথে চলে যাচ্ছে কি না।
আমাদের গার্ডরেইল ফাংশনটি এআই-এর জন্য ঠিক সেই **'বডিগার্ড'** বা **'দারোয়ান'** হিসেবে কাজ করে।

## 🧠 গাণিতিক ধারণা (Math Intuition)

নিরাপত্তা নিশ্চিত করার একটি সহজ উপায় হলো **Cosine Similarity**।
আমরা যদি জানি "বোমা" বা "মারামারি" এই শব্দগুলোর ভেক্টর স্পেসে অবস্থান কোথায়, তবে আমরা ইনপুট ভেক্টরের সাথে ডিস্টেন্স মেপে বুঝতে পারি সেটি কতটা ক্ষতিকর।
$$Similarity = \cos(\theta) = \frac{A \cdot B}{||A|| ||B||}$$
যদি সিমিলারিটি স্কোর বেশি হয় (যেমন- ০.৯), তবে আমরা উত্তরটি ব্লক করে দিই।

## 💻 কোড ইমপ্লিমেন্টেশন (Building a Guardrail Layer)

```python
# কোনো API ছাড়া সিম্পল রুল-বেসড সেফটি ফিল্টার
class SafetyGuard:
    def __init__(self):
        # ক্ষতিকর শব্দের ডেটাবেস
        self.banned_words = ["বিস্ফোরক", "মারামারি", "চুরি", "হ্যাকিং", "bomb"]
        
    def check_input(self, text):
        for word in self.banned_words:
            if word in text.lower():
                return False, f"⚠️ ক্ষমা করবেন, আমি '{word}' সংক্রান্ত কোনো উত্তরের জন্য প্রোগ্রাম করা নই।"
        return True, text

    def check_output(self, response_text):
        # আউটপুটেও সেম লজিক বা আরও কড়া নিয়ম
        if len(response_text) < 2:
            return "দুঃখিত, আমি ঠিক বুঝতে পারছি না।"
        return response_text

# ৩. টেস্টিং
guard = SafetyGuard()

user_question = "কীভাবে বোমা বানাবো?"
is_safe, message = guard.check_input(user_question)

if not is_safe:
    print(message)
else:
    # এখানে আপনার মডেলের জেনারেশন ফাংশন হতো
    print("AI Response: [Generated text...]")
```


> [!IMPORTANT]
> **Output Validation:** শুধু ইনপুট চেক করলেই হবে না, অনেক সময় মডেল ভুল করে আউটপুটে ক্ষতিকর কিছু বলে দিতে পারে। তাই 'Post-processing' বা আউটপুট ভ্যালিডেশন সমানভাবে জরুরি।
