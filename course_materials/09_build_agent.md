# Module 9: স্ক্র্যাচ থেকে এআই এজেন্ট (Building AI Agent)

আপনি চ্যাটবট বানিয়েছেন, কিন্তু চ্যাটবট শুধু কথা বলতে পারে। **AI Agent** হলো সেই সত্তা যে কথা বলার পাশাপাশি নিজে কাজ করতে পারে (Action নিতে পারে)। আজ আমরা কোনো ফ্রেমওয়ার্ক (যেমন- LangChain) ছাড়া নিজে এজেন্ট লুপ বানাবো।

## ১. এজেন্ট কী?
এজেন্ট হলো: **Model + Thinking Loop + Tools**।
সে নিচের ধাপগুলো মেনে কাজ করে (ReAct Pattern):
1.  **Thought:** ব্যবহারকারীর লক্ষ্য বুঝবে এবং কী করতে হবে তা ভাববে।
2.  **Action:** একটি টুল সিলেক্ট করবে (যেমন- ক্যালকুলেটর)।
3.  **Observation:** টুলের ফলাফল দেখবে।
4.  **Final Answer:** ব্যবহারকারীকে উত্তর দেবে।

---

## 💡 বাস্তব জীবনের উদাহরণ (Analogy)

ভাবুন আপনি একজন অন্ধ মানুষের সহকারী।
ব্যবহারকারী বলল: "বাইরে কি বৃষ্টি পড়ছে?"
১. **Thought:** আমাকে জানালা দিয়ে দেখতে হবে।
২. **Action:** জানালা খুললেন।
৩. **Observation:** বাইরে রোদ দেখা যাচ্ছে।
৪. **Final Answer:** আপনি বললেন, "না, বাইরে রোদ।"

এজেন্টও এভাবেই টুল ব্যবহার করে রিয়েল ওয়ার্ল্ড ডেটা সংগ্রহ করে।

---

## 🧠 গাণিতিক ধারণা (Math Intuition)

এজেন্টের কোনো আলাদা সূত্র নেই, এটি মূলত একটি **Recursive Function**।
$Answer = Agent(Goal, State, Tools)$
মডেলটি বারবার একই লুপে ঘোরে যতক্ষণ না সে কাঙ্ক্ষিত আউটপুট পায়।

## 💻 কোড ইমপ্লিমেন্টেশন (The ReAct Loop)

```python
# কোনো ফ্রেমওয়ার্ক ছাড়া একটি সিম্পল এজেন্ট লুপ
import time

def get_answer(model_output):
    # এটি মডেলের টেক্সট থেকে Action খুঁজে বের করে
    if "CALC:" in model_output:
        return "ACTION", model_output.split("CALC:")[1]
    return "FINAL", model_output

def agent_loop(user_goal):
    print(f"লক্ষ্য: {user_goal}\n")
    
    # ১. Thought (মডেলকে প্রম্পট দিচ্ছি)
    thought = "আমাকে যোগফল বের করতে হবে। আমি ক্যালকুলেটর ব্যবহার করব।"
    print(f"এজেন্ট ভাবছে: {thought}")
    
    # ২. Action (মডেল লজিক্যালি Action চালাল)
    action_type, action_val = get_answer("CALC: 5 + 10")
    
    if action_type == "ACTION":
        print(f"এজেন্ট অ্যাকশন নিচ্ছে: {action_val}")
        
        # ৩. Observation (টুলের রেজাল্ট)
        observation = eval(action_val) # সিম্পল ক্যালকুলেটর টুল
        print(f"এজেন্ট দেখল ফলাফল: {observation}")
        
        # ৪. Final Answer
        final_thought = f"আমি উত্তর পেয়েছি। ৫ এবং ১০ এর যোগফল হলো {observation}।"
        print(f"\nফাইনাল উত্তর: {final_thought}")

agent_loop("৫ এবং ১০ এর যোগফল কত?")
```

## 📊 আউটপুট উদাহরণ (Output)

```text
লক্ষ্য: ৫ এবং ১০ এর যোগফল কত?

এজেন্ট ভাবছে: আমাকে যোগফল বের করতে হবে। আমি ক্যালকুলেটর ব্যবহার করব।
এজেন্ট অ্যাকশন নিচ্ছে: 5 + 10
এজেন্ট দেখল ফলাফল: 15

ফাইনাল উত্তর: আমি উত্তর পেয়েছি। ৫ এবং ১০ এর যোগফল হলো 15।
```

> [!IMPORTANT]
> **No LangChain Rule:** আপনি যদি নিজে এই লুপ কন্ট্রোল করতে পারেন, তবে আপনি যেকোনো কাস্টম এনভায়রনমেন্টে নিজের এআই সিষ্টেম ডিজাইন করতে পারবেন।
