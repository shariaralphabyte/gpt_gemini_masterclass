# Module 13: সেমিস্টার প্রজেক্টস (Projects)

অভিনন্দন! আপনি কোর্সের শেষ প্রান্তে চলে এসেছেন। এখন সময় হয়েছে আপনার শেখা জ্ঞানকে বাস্তবে রূপ দেওয়ার। এই মডিউলে আমরা ৩টি বড় প্রজেক্টের পরিকল্পনা এবং গাইডলাইন দেখবো।

## ১. প্রজেক্ট ১: ছোট বাংলা এলএলএম ট্রেইন করা
আমরা মডিউল ৪, ৫ এবং ৬-এর জ্ঞান ব্যবহার করে একটি ছোট ১-লেয়ারের ট্রান্সফর্মার মডেল ট্রেইন করবো যা বাংলা বাক্য শেষ করতে পারবে।
*   **কাজ:** একটি ছোট বাংলা টেক্সট ফাইল লোড করা, বিপিই টোকেনাইজার বানানো এবং ৫-১০ মিনিট ট্রেনিং চালানো।

## ২. প্রজেক্ট ২: সেফটি লেয়ারসহ চ্যাটবট
মডিউল ৮ এবং ১১-এর সমন্বয়।
*   **কাজ:** এমন একটি চ্যাটবট বানানো যা ব্যবহারকারীর সাথে কথা বলবে, কিন্তু কোনো ‘নিষিদ্ধ’ প্রশ্ন করলে তাকে ব্লক করে দেবে।

## ৩. প্রজেক্ট ৩: কাস্টম এআই এজেন্ট (Final Boss)
মডিউল ৯ এবং ১০-এর সমন্বয়।
*   **কাজ:** একটি এজেন্ট বানানো যার মেমোরি থাকবে এবং সে নিজে ক্যালকুলেটর টুল ব্যবহার করে জটিল গণিত সমাধান করে আপনাকে উত্তর দেবে।

## 🧠 গাণিতিক ধারণা (Math Intuition)

প্রজেক্টের সময় আপনার মূল ফোকাস থাকবে **Generalization** এর ওপর।
আপনার মডেল কি শুধু মুখস্থ বলছে নাকি সে আসলেই প্যাটার্ন বুঝেছে? আপনি এটি মাপতে পারেন এই সূত্র দিয়ে:
$$Perplexity = \exp(\text{Cross Entropy Loss})$$
পারপ্লেক্সিটি যত কম হবে, আপনার মডেল তত ভালো প্রেডিকশন করতে পারছে।

## 💻 কোড ইমপ্লিমেন্টেশন (Integrated Project Blueprint)

# এটি একটি মাস্টার ক্লাস যা সব মডিউলকে এক করে
class ProjectMasterAI:
    def __init__(self, model, tokenizer, guard):
        self.model = model
        self.tokenizer = tokenizer
        self.guard = guard
        self.memory = []

    def run_system(self, user_input):
        # ১. সেফটি চেক (Module 11)
        is_safe, msg = self.guard.check_input(user_input)
        if not is_safe: return msg
        
        # ২. থিংকিং লুপ (Module 9)
        print("Thinking...")
        
        # ৩. টুল ইউজ (Module 10)
        if "কত" in user_input:
            result = "১২৫" # ধরুন টুল ক্যালকুলেট করেছে
            return f"মডেল নিশ্চিত করেছে ফলাফল: {result}"
            
        # ৪. জেনারেশন (Module 8)
        return "আপনার প্রজেক্ট সফলভাবে কাজ করছে!"
