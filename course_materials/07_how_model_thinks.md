# Module 7: মডেল কীভাবে চিন্তা করে? (Inside the LLM Flow)

আমরা অনেক কোড করেছি, কিন্তু একটি এলএলএম (LLM) ঠিক কোন কোন ধাপ পেরিয়ে একটি উত্তর তৈরি করে? আজ আমরা সেই ‘থট প্রসেস’ বা ডাইগ্রামটি দেখবো।

## ১. ইনফারেন্স ফ্লো (The Inference Pipeline)
যখন আপনি একটি মেসেজ টাইপ করেন, তখন মডেল নিচের ধাপগুলো পার করে:

1.  **Input Text:** আপনার টাইপ করা বাক্য।
2.  **Tokenization:** বাক্যটিকে সংখ্যায় রূপান্তর।
3.  **Embedding:** সংখ্যার বদলে একটি ভেক্টর বা ডিজিটাল কোঅর্ডিনেট সেট করা।
4.  **Attention Blocks:** বাক্যটির বিভিন্ন শব্দের মধ্যে সম্পর্ক বোঝা।
5.  **Logits:** পরবর্তী সম্ভাব্য সব শব্দের জন্য একটি স্কোর তৈরি করা।
6.  **Softmax:** স্কোরগুলোকে পার্সেন্টেজে রূপান্তর করা।
7.  **Sampling:** সেরা শব্দটি বেছে নেওয়া।

---

## 💡 বাস্তব জীবনের উদাহরণ (Analogy)

ভাবুন আপনি একটি রেস্টুরেন্টে গিয়ে খাবার ‘অর্ডার’ দিচ্ছেন।
১. **Input:** আপনি মেনু দেখে অর্ডার দিলেন।
২. **Tokenization:** ওয়েটার সেটি তার প্যাডে কোড আকারে লিখলেন।
৩. **Embedding:** কুক সেই কোড দেখে বুঝলেন এটা মশলাদার হবে নাকি মিষ্টি।
৪. **Attention:** কুক দেখলেন আপনি ‘চিকেন’ চেয়েছেন এবং পাশে ‘ঝাল’ লিখেছেন; তাই তিনি চিকেনের সাথে লঙ্কার সম্পর্ক বুঝলেন।
৫. **Output:** ফাইনাল ডিশ আপনার টেবিলে আসলো (টোকেন জেনারেশন)।

## 🧠 গাণিতিক ধারণা (Math Intuition)

মডেলের লাস্ট লেয়ার একটি **Probability Distribution** দেয়:
$$P(x_i | x_{<i}) = \text{softmax}(Z_i)$$

এখানে $Z_i$ হলো প্রেডিকশন লেয়ারের আউটপুট। যদি ভোকাবুলারিতে ৩টি শব্দ থাকে: [আম, জাম, আমলকী], এবং আউটপুট আসে [১০, ১, ০], তবে সফটম্যাক্স করলে "আম" আসার সম্ভাবনা হবে প্রায় ৯৯%।

## 💻 কোড ইমপ্লিমেন্টেশন (The "Thought" Simulator)

```python
import torch

# ১. ড্রামি ভোকাবুলারি
vocab = {0: "i", 1: "love", 2: "coding", 3: "ai"}

# ২. ধরুন লাস্ট লেয়ার থেকে আসা স্কোর (Logits)
logits = torch.tensor([1.5, 0.2, 5.8, 2.1])

# ৩. সফটম্যাক্স (যাতে আমরা পার্সেন্টেজ পাই)
probs = torch.softmax(logits, dim=0)

print("ভোকাবুলারি অনুযায়ী প্রতিটি শব্দের সম্ভাবনা:")
for i, p in enumerate(probs):
    print(f"{vocab[i]}: {p.item()*100:.2f}%")

# ৪. সেরা শব্দটি বেছে নেওয়া (Sampling)
next_token_id = torch.argmax(probs).item()
print(f"\nমডেলের সিদ্ধান্ত: পরবর্তী শব্দ হবে '{vocab[next_token_id]}'")
```
