## ১. AI vs ML vs Deep Learning

অনেকেই এই তিনটি জিনিস গুলিয়ে ফেলেন। সহজভাবে বুঝুন:

*   **Artificial Intelligence (AI):** যেকোনো সিস্টেম যা মানুষের মতো চিন্তা করতে বা সিদ্ধান্ত নিতে পারে। 
*   **Machine Learning (ML):** এআই-এর একটি সাবসেট, যেখানে ডেটা দেখে প্যাটার্ন শেখানো হয়।
*   **Deep Learning (DL):** এমএল-এর একটি শাখা যা 'Neural Network' ব্যবহার করে জটিল কাজ সমাধান করে (যেমন- চ্যাটজিপিটি)।

## 💡 বাস্তব জীবনের উদাহরণ (Analogy)

মনে করুন আপনি আপনার ছোট ভাইকে 'আম' এবং 'জাম' চেনাতে চাচ্ছেন।
*   **AI:** যখন সে ফল দেখে বলতে পারে এটা আম বা জাম।
*   **ML:** আপনি তাকে অনেকগুলো আমের ছবি দেখালেন এবং বললেন আমের গায়ের রঙ হলুদ বা সবুজ হয়। সে প্যাটার্ন শিখল।
*   **Deep Learning:** আপনি তাকে শুধু ফলগুলো দিলেন, সে নিজে থেকেই ফলের গঠন, গন্ধ এবং রঙের গভীর প্যাটার্ন বুঝে আম ও জাম আলাদা করে ফেলল।

## 🧠 গাণিতিক ধারণা (Math Intuition)

এআই-এর মূল একক হলো **Neuron**। এটি একটি সাধারণ গাণিতিক ফাংশন মাত্র:

$$y = f(Wx + b)$$

*   **x:** ইনপুট ডেটা (যেমন- আমের ওজন)।
*   **W (Weight):** গুরুত্ব। আমের জন্য ওজনের গুরুত্ব কতটুকু?
*   **b (Bias):** অফসেট বা সংশোধন।
*   **f (Activation):** এই ফলাফলটিকে আমরা সচল করব কি না (যেমন- ReLU বা Sigmoid)।

## 💻 কোড ইমপ্লিমেন্টেশন (NumPy)

আসুন কোনো ফ্রেমওয়ার্ক ছাড়া শুধু NumPy দিয়ে একটি সরল নিউরন তৈরি করি।

```python
import numpy as np

# ১. Activation Function (ReLU)
# এটি নেগেটিভ ভ্যালুকে ০ করে দেয়, পজিটিভকে যেমন আছে তেমন রাখে।
def relu(x):
    return np.maximum(0, x)

# ২. ইনপুট (Input)
inputs = np.array([1.2, 0.5]) # ধরুন ফলের ওজন এবং লালচে ভাব

# ৩. ওয়েটস এবং বায়াস (Weights & Bias) - র্যান্ডমভাবে শুরু
weights = np.array([0.8, -0.3])
bias = 0.5

# ৪. ক্যালকুলেশন (Weighted Sum)
# z = (W1*x1 + W2*x2) + b
weighted_sum = np.dot(inputs, weights) + bias

# ৫. সচল করা (Activation)
output = relu(weighted_sum)

print(f"লজিক্যাল সাম: {weighted_sum}")
print(f"নিউরাল নেটওয়ার্কের আউটপুট: {output}")
```

## 📊 আউটপুট উদাহরণ (Output)

```text
লজিক্যাল সাম: 1.31
নিউরাল নেটওয়ার্কের আউটপুট: 1.31
```

> [!TIP]
> **Loss Function:** আমাদের মডেল ভুল করলে আমরা এই ফাংশন দিয়ে তা মাপি। যেমন: **MSE (Mean Squared Error)**।
> **Optimizer:** ভুলের পর ওয়েটস আপডেট করার জন্য আমরা **Adam** বা **SGD** ব্যবহার করি। আমরা মডিউল ৩-এ এটি স্ক্র্যাচ থেকে বানাবো।
